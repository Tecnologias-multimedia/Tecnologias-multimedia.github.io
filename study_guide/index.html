1 Methodology

TM (Tecnologías Multimedia) follows the PBL (Project-Based Learning) methodology. The students, helped by the teacher, develop a project during the course. This project is InterCom(municator), an application that allows us to communicate networked users throught the Internet.

The project is implemented as a sequence of milestones. The students, in groups of up to 4 people, propose solutions for each milestone. Such solutions are presented to the rest of the class by each group, and each group gives and receives a score and feedback from the rest of the class and also from the teacher.

2 Milestones

  1. OS (Operating System) Provisioning.
  2. Installation and basic programming with Python.
  3. Git, GitHub and the Fork-and-Branch Git Workﬂow.
  4. ‘Wiring” the ADC with the DAC and measuring latencies.
  5. Minimal InterCom.
  6. Impact of the jitter.
  7. Impact of the bit-rate.
  8. Buﬀering.
  9. Compressing the audio data with zlib.
  10. Bit-rate Control through Quantization.
  11. Inter-channel decorrelation in stereo audio signals.
