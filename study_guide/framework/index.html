1 Description

The InterCom project [7] is a collection of Python modules [3]. Python has been ported to almost all the current OSs, including mobile devices. Among all available development environments, we will develop the project on Linux, and more concretely, on Xubuntu 22.04 (Jammy Jellyfish) [5], running natively (no virtualization), if possible. Please, follow for this, to install Xubuntu. Alternatively, you can use any other Linux distribution (or even, any other OS), but you should be able to provide technical support yourself, in case you need it.

You don’t need to master Python to follow this course, but it is convenient for you to rely on some Python programming tutorial, such as The Python Tutorial [2] if you realize that the language is a setback for you. If you need to start with Python from scratch, an introduction to Python such as this workshop of YAPT [4] could also be helpful. See also ZetCode’s Python Tutorial. Follow this to install Python. Alternatively (but reducing the chances of solving any possible issue), you can use the Python interpreter shipped with your OS. In this case, it is strongly recommended to use an specific Python environment for the InterCom project.

Finally, to work in the InterCom project [7] you must understand the basics of Git [1] and GitHub1 [8], and how to use the The GitHub (Work-)Flow and the Fork-and-Branch Git Workflow [6]. Be aware that to contribute to InterCom, an GitHub account is required. Please, follow this minimal Git guide. In the guide, you must consider that the <organization_name> is Tecnologias-multimedia and that the <repo_name> is intercom.
