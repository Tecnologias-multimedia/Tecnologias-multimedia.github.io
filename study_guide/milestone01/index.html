1 Description

The InterCom project [4] is written in Python [3]. You will need to understand the syntax and how to develop/run Python programs (modules and packages).

2 What you have to do?

  1. You don’t need to master Python to follow this course, but it is convenient for you to follow a tutorial [1] if you realize that the language is a setback for you. If you need to start from scratch with Python, a workshop [2] like this could be interesting.

3 Timming

There is not limit of time for ﬁnishing this milestone. Develop it at your own pace.

4 Deliverables

None.
