Tecnologías Multimedia - Study Guide - Milestone 1: Git and GitHub

Vicente González Ruiz - Depto Informática - UAL

September 7, 2020

1 Description

To work in the InterCom project [6] you will need to understand the basics of Git and GitHub.

2 What you have to do?

  1. Have a look the Git [3, 2] and the GitHub [4] websites. If you don’t have a GitHub account, please, create one.
  2. Create a test project at GitHub using your account. See [1].
  3. See The Fork and Branch Git Workﬂow [5].
  4. Make a fork of the InterCom [6] project.

3 Timming

You should reach this milestone at most in one week.

4 Deliverables

None.
