1 Description

1.1 Overlapped transforms for minimizing the distortion

Transform coding implies to split the signal into chunks, and to compute the transform of each chunk. When the output coeﬃcients are quantized, it is possible that signiﬁcative (and unpleasant) distortions may appear between the border samples of the chunks (see this notebook).

One solution for the current (i-th) chunk is to use the last samples of the previous (i − 1-th) chunk and the ﬁrst samples of the next (i + 1-th) chunk for computing the transform of the current (i-th) chunk. Therefore, to avoid signal discontinuities between chunks, we can overlap them. This idea has been implemented in this notebook.
